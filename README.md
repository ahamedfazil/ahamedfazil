# 💫 About Me:
Senior Full Stack Engineer focused on AI engineering. Most of my career has been inside financial institutions - banks and a trading platform - which shaped how I approach AI: it has to be tested, monitored, governed, and still working six months later.<br><br>I build production GenAI systems on Azure - RAG pipelines over enterprise data, agentic workflows with Semantic Kernel and LangChain, and Azure API Management as an AI gateway for LLM traffic. I've also rolled out GitHub Copilot and Claude across engineering teams, wired agentic coding into the SDLC, and built evaluation suites so agents don't quietly break in production.<br><br>Before the AI work: cloud and data engineering and extensive SharePoint/M365 platform development. I write about that at [sharepointpals.com](https://www.sharepointpals.com) and contribute to the Microsoft PnP community.<br><br>- 🏦 Domain: banking, financial services, trading platforms<br>- 🔭 Currently: agentic AI applications on Azure AI Foundry, with MCP integrations<br>- 🤖 Agentic SDLC: Claude Code, GitHub Copilot, evaluation-driven agent workflows<br>- 🧰 Stack: TypeScript, React, Python · Azure OpenAI, AI Search, Container Apps, Cosmos DB<br>- ✍️ Blog: [sharepointpals.com](https://www.sharepointpals.com)
